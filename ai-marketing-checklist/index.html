1) Plan & Research

  • Define ICP and Jobs-to-Be-Done. ICP discovery prompt
  • Clarify offer & value props (pain → promise → proof).
  • Keyword set: primary + supporting + questions.
  • Channel map: blog, social, email, partners, communities.
  • Set KPIs (leads, MQLs, CTR, CPL, demo rate, revenue).

2) Produce Content

  • Create pillar + clusters. Use Automate Content AI to generate briefs & outlines.
  • Write with clear H1/H2, FAQs and internal links.
  • Generate social snippets & carousels from each post.
  • Design hero & OG images (brand + benefit).

Helpful resources: Marketing Templates, Marketing Prompts, Prompt Cheat-Sheet.

3) Build Funnels

  • Lead magnet: checklist, mini-guide, calculators.
  • Landing page: promise, proof, preview, CTA.
  • UTM conventions + analytics events.
  • Thank-you page with next action (demo, community).

See: AI Funnel Automation.

4) Email & Nurture

  • Welcome + 3-email value sequence with case/guide.
  • Behavioural triggers: view pricing, download, reply.
  • Plain-text sales handoff for hot leads.
  • Weekly content recap with 1 CTA.

Ready sequences: AI Email Prompts.

5) Sales Enablement

  • Objection handling library (price, timing, features).
  • Discovery → demo prompt set for reps.
  • Follow-up rhythm (1-2-5-10) with value nuggets.

Tools: AI Sales Tools.

6) Measure & Optimize

  • Dashboard: traffic → leads → MQL → revenue.
  • A/B tests on headlines, hero, offers and emails.
  • Content refresh every 60–90 days (FAQs + links).
